HOWTO: Internet access using ConnMan-GTK

Author: user1234 Publishing date: 24.11.2022 : UTC

What is ConnMan-GTK?

ConnMan-GTK is a GUI (Graphical User Interface) application which is used in all latest puppies to connect to the internet and bluetooth.

How to open up ConnMan-GTK?

Open up Application Menu > Puppy Setup > Internet. Otherwise, run this command in a terminal:-

connman-gtk
The opened window should look somehting like this:-

What if ConnMan-GTK is not present on my Puppy?

ConnMan-GTK should probably be present on all puppies subsequent to Fossa64. But in case it is not present, then you should try reading HOWTO: Internet page for other available options for connecting to internet.

How to connect to Wireless Network?

Follow these steps to connect to a wirless network using SNS:-

  1. Open up ConnMan-GTK (Read how to open up ConnMan-GTK).
  2. Go to Wireless tab in the ConnMan-GTK window (if not already opened in that tab).
  3. The window should now look like this:-

  4. The button present at the top-right of the window can be used to enable/disable WiFi.
  5. Now click on any one of the listed wireless networks and then click on the Connect button.
  6. You will now be prompted to enter password for the wireless network you want to connect to. Enter the password and click OK button.

  7. Wait for few seconds to get connected to the internet. Now enjoy surfing on the internet!

How to connect to Wired (Ethernet) Network?

Follow these steps to connect to a wirless network using SNS:-

  1. Open up ConnMan-GTK (Read how to open up ConnMan-GTK).
  2. Go to Wired tab in the ConnMan-GTK window (if not already opened in that tab).
  3. If you have just a single Ethernet connected, then you may be automatically connected to that network. If not, then click on any of the listed wired networks to select them; and then click Connect button:-

  4. Wait for few seconds to get connected to the internet. Now enjoy surfing on the internet!
